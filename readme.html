Opportunity Finder - Dashboards

Executive maintenance dashboards built to identify additional revenue opportunities, operational insight, and data-quality gaps.

Purpose

This repository hosts a set of interactive dashboards built from CMMS work-order, asset, and location data. They are published here so they can be opened in any browser with no Power BI, no server, no login, and no software to install.

The dashboards turn raw work-order exports into an executive view focused on the core question:

Where can Camelot generate additional revenue beyond baseline contracts?

The dashboards highlight opportunities to self-perform work currently outsourced to vendors, recover missed billing, and identify project-bid candidates, while also surfacing data-quality, geography, and SWOT insights.

Each page includes live Client / Database / Month (or State) filters, KPI cards, trend and breakdown charts, auto-generated insights, and sortable detail tables. Every file is fully self-contained with all data and charts embedded, and no external or CDN calls.

Dashboards

Master Executive Dashboard Primary starting point and executive overview. Opportunity Finder Revenue opportunities, outsourced work, and project-bid candidates. Work Orders Work-order trends, categories, clients, vendors, and detail tables. Assets Asset records, condition insight, and maintenance visibility. Locations Site directory and geography across client portfolios — address, city/state, and property managers. Data Quality Missing, inconsistent, or incomplete records that need cleanup. Data Quality Scorecard Per-account and per-database data-capture grades (0-100).
Start with the Master Executive Dashboard, then use the in-page tabs to move between views.
